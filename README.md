## 简介

一个简单的证书帮助工具

## 安装

### Maven

在项目的 `pom.xml` 的 `dependencies` 中加入以下内容:

```xml

<dependency>
  <groupId>io.github.zornx5</groupId>
  <artifactId>certificate-helper</artifactId>
  <version>0.2.0</version>
</dependency>
```

### Gradle

```gradle
implementation 'io.github.zornx5:certificate-helper:0.2.0'
```
